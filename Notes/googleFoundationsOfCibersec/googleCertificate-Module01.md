### 2. [What is Cybersecurity?]

- Cybersecurity ensures confidentiality, protects both the user and the organization. Data losses and leaks affect both 
- **Core Concepts:**
 1.  Phishing -> the practice of impersonate trusted entities or individuals to deceive people into reveal sensitive information
 2.  Compliance -> adhering to standards and regulations
 3.  Security Controls -> safeguards to reduce specific risks
 4.  Security Frameworks -> guidelines to build plans that mitigate risks to data
 5.  Security posture -> Organization ability to manager its defense or assets
 6.  Threat actor -> malicious attacker
 7.  Internal threat -> trusted partner who represents a security risk(not necessarily willingly)
 8.  Network Security -> Protect Network infrastructure of a company
 9.  Cloud Security -> Protect Cloud infrastructure of a company
 10.  SIEM -> Security information and event management tools
 11.  Computer Forensics -> Analysis of a machine after an attack or error in the system
 12.  PII/SPII -> Personally identifiable information and Special Personally identifiable information are data that can lead to the inferring of an individual's identity.

 PII is all that info that exposes the identity of someone while SPII are specific information that exposes sensitive data
